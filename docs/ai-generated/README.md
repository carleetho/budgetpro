# AI Generated Reports (Point-in-time)

Este directorio agrupa reportes generados por IA para analisis operativo y planeacion.

Reglas de uso:
- Son snapshots puntuales, no especificaciones canonicas de dominio.
- No reemplazan `docs/canonical/modules/*_MODULE_CANONICAL.md`.
- Deben interpretarse junto con evidencia de codigo y pruebas.

Objetivo:
- Reducir ruido en la raiz del repo.
- Evitar que agentes automaticos consuman estos archivos como fuente normativa principal.
